typora 的markdown语法

段落与换行符

段落只是一行或多行连续的文本。在 markdown 源代码中，段落由多个空行分隔。在Typora中，您只需按下 enter 即可创建新段落。

按**Shift + return** 可创建单个换行符。但是，大多数 markdown 解析器将忽略单行中断，要使其他 markdown 解析器识别您的换行符，可以在行尾留下两个空格，

enter 新段落 shift+enter 新行

typora在每个语法后面都需要加空格才能生效，且一般还要加上enter 产生效果

typora 快捷键为ctrl + 1(表示一级标题) ctrl+2 (表示二级标题) 依次类推

typora 使用ctrl + = (在键盘上也是+的键)用来提升标题等级 ctrl + -用来降低标题等级

typora 中的ctrl键就类似于idea中的alt键, 是绝大多数快捷键的组合键 , 非常重要

标题

标题在行的开头使用1-6个＃字符，对应于标题级别1-6。在typora中，按下回车键就能展示效果

代码块

输入```键之后输入一个可选的语言标识符，然后按enter键后输入代码，我们将通过语法高亮显示它：

使用``` + 语言标识符 即可创建相应的代码块

typora 中使用 ctrl + shift + k 也可产生代码块 (一般般)

表格

输入 | First Header | Second Header | 并按下 return 键将创建一个包含两列的表。

创建表后，焦点在该表上将弹出一个表格工具栏，您可以在其中调整表格，对齐或删除表格。您还可以使用上下文菜单来复制和添加/删除列/行。
